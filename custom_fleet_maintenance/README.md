# Custom Fleet Maintenance

Module de gestion des maintenances preventives et curatives pour le parc automobile Odoo 19.

- Creation de commandes d'achat depuis l'intervention avec consolidation des couts reels.
- Transfert de pieces stockables vers l'atelier via picking interne directement depuis l'intervention.
- Assistant backend pour les conducteurs afin de declarer une panne (menu Maintenance > Declarer une panne).
